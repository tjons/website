What I'm doing now

Last updated Sunday, October 26th, 2025

It has been a really busy year, and I haven't been able to write as much as I planned. Currently, I am at home after a week traveling to Tucson, Arizona for work. Kubecon is coming up, and so I am procrastinating on my talk prep and have completely rebuilt this website's hosting infrastructure this morning.

I'm reading Fall; or, Dodge in Hell, by Neal Stephenson. It is a great book, and I am getting excited about SciFi for the first time in a while. For technical reading, I am slowly working my way through A Practical Guide to TPM 2.0, which is available as a free eBook. I've spent a lot of time this year thinking about and working with TPMs.

This website is now served to you by a Hetzner Cloud VM running Fedora 42, in Ashburn, VA. The server is listening on a port served by an Envoy Proxy service, configured with systemd. The website contents are served to you by an nginx docker container, also configured by systemd. I built Envoy from source for this successfully, after many attempts. The server is managed by Terraform and the website is configured by an Ansible playbook. The website continues to live as a git repo, with a systemd timer that reloads the git repository every 5 minutes. If you see this, it worked!
